# Trading Analytics System

Sistema completo de análisis de trading con:
- Daily Plan (planificación pre-mercado)
- Parser (entrada de trades)
- Dashboard (5 vistas de análisis)

## Versión Actual
v0.1.0-prototype (Visual prototype)

## Estructura
```
TradingSystem/
├── docs/          # Documentación
├── src/           # Código fuente
├── assets/        # Recursos
├── tests/         # Tests
└── backups/       # Respaldos
```

## Cómo usar
1. Abrir `src/daily-plan.html` en navegador
2. Llenar plan del día
3. Abrir `src/parser.html` para registrar trades
4. Abrir `src/dashboard.html` para análisis

## Stack Tecnológico
- HTML5
- CSS3 (Tailwind CDN)
- JavaScript ES6+
- window.storage API
